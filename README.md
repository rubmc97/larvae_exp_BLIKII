# Bioinformatic processing of 16S rRNA amplicon and long-read metagenomics datasets from freshwater larval gut bacterial communities

In the present repository you can find the code used to process and analyse the mentioned dataset.
Contains:
-larvae_exp_script_bash.Rmd : R markdown containing the bash code in chunks, whose tasks were ran using SLURM in a HPC cluster.
-16S_analyses_script_BLIK_III: R script containing the R code to reproduce the results of the 16S rRNA amplicon dataset part.
